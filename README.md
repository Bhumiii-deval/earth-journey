# Carbon Footprint Awareness Platform

## Overview

This project helps users understand and track their carbon footprint based on daily lifestyle choices.

## Features

* Carbon footprint calculator
* Travel, electricity, and food impact analysis
* Dashboard with charts
* Sustainability tips
  

## 🧪 Testing

The application has been manually tested with multiple user inputs to ensure correct calculations and stable performance.

### Test Cases:

* Short distance travel input (0–5 km)
* Medium distance travel input (10–50 km)
* High electricity usage input
* Different transport modes (car, bus, train, bike, walking)
* Food preference variations

### Validation:

* Ensures no empty inputs break the calculator
* Handles different numeric ranges smoothly
* Provides correct category output (Low / Medium / High)

All features were tested in a browser environment for responsiveness and correctness.

## How it works

Users enter their daily activities like travel distance, transport type, electricity usage, and food habits. The system calculates a carbon footprint score and shows impact level (Low, Medium, High).
